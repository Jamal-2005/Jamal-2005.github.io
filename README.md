# Syed Jamal Ur Rahman — Portfolio

Personal portfolio website for a Data Engineer, built with vanilla HTML, CSS, and JavaScript.

## 🚀 Live Site

Deploy via GitHub Pages: `https://<your-username>.github.io/<repo-name>/`

## 📁 Project Structure

```
portfolio/
├── index.html      # Main HTML file
├── style.css       # Styles (CSS variables, layout, animations)
├── script.js       # Interactivity (scroll, nav, fade-ins)
└── README.md       # This file
```

## ✨ Features

- Fully responsive (mobile, tablet, desktop)
- Dark minimal aesthetic with warm accent tones
- Smooth scroll-triggered animations
- Sticky nav with active section highlighting
- Mobile hamburger menu

## 🛠 Deployment (GitHub Pages)

1. Create a new GitHub repository (e.g., `portfolio`)
2. Push all files to the `main` branch
3. Go to **Settings → Pages**
4. Set Source to `Deploy from a branch` → `main` → `/ (root)`
5. Save — your site will be live in ~60 seconds

## 🔧 Customization

- Update the LinkedIn URL in `index.html` (search for `YOUR-LINKEDIN`)
- Update GitHub project links when you publish repos
- Replace phone/email if needed
